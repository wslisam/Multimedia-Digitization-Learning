This tool is designed to help you understand how computers take real-world sights and sounds and turn them into digital data (0s and 1s). 

## 🌟 How to Use This App for Learning

This app is divided into a "Learning Path" and "Tools & Concepts". We recommend going through the tabs in order from top to bottom.

### 1. Introduction
Start here! Learn the fundamental concept of **Digitization**. Pay close attention to the two main steps:
*   **Sampling:** Taking snapshots.
*   **Quantization:** Rounding values to fit available bits.
*   *Teacher's Tip:* Use the "Think & Discuss" box at the bottom to reflect on what might happen if we don't take enough snapshots.

### 2. Audio Digitization
Interact with the sliders to see how sound is digitized!
*   **Sampling Rate Slider:** Move this to see how taking more snapshots (slices) per second makes the digital wave follow the original red wave closer horizontally.
*   **Bit Depth Slider:** Move this to see how adding more "levels" makes the vertical steps smaller and more accurate.
*   *Self-Check:* Can you make the green digital wave look almost exactly like the red analog wave? What settings did you need?

### 3. Image Digitization
See how pictures are turned into pixels.
*   **Resolution Slider:** Drag this to change the grid size. Notice how a low resolution looks "pixelated" (blocky), while a high resolution looks clear. 
*   Read the section on **Color Depth** to understand how many colors a computer can display based on the number of bits used per pixel.

### 4. File Size Calculator
Now for the math! Computers need storage space for all these bits.
*   Try changing the image dimensions (Width/Height) or the Audio Duration. Watch how fast the file size grows!
*   *Challenge:* Try calculating the size of a 3-minute (180 seconds) song at CD quality (44100 Hz, 16-bit, Stereo). Why is it so big?

### 5. File Formats & Compression
Because uncompressed files are huge, we need compression.
*   Learn the difference between **Lossless** (perfect quality, but bigger files) and **Lossy** (lower quality, but tiny files).
*   Review the common file formats (like JPG, PNG, MP3) so you know which one to choose for your next project!

## 🌐 Language Support
Click the language button in the top right corner (`English` / `中文`) at any time to switch the entire app's language. This is great if you are learning computer science terms in both languages!
